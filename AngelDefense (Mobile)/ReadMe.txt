For Project File
Please noted that Switch to Android Platform and Maximize on Screen are recommended when playing in Unity   

For Android built
The game has many touchscreen buttons to touch. 
> Y	: Attack
> X	: Change Weapon
> B	: Run
> A 	: Blocking and Cancel the player Run

Also, other touchable buttons such as
> Arrows 				: Disable the blocks  
> Stones  				: Activate barrier usage (by default)
> Four toggles at the right corner	: Barrier  1, Barrier 2, Barrier 3 & Barrier 4
> Blue circle with white square		: Paused the game and Hints
